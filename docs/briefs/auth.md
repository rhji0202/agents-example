# Auth Brief

## Summary

고객(개인/사업자)과 직원이 계정을 관리하고, 관리자가 고객·직원·권한·포인트를 통합 관리한다.

## Flow

### 회원가입/로그인

1. 고객이 회원가입 (아이디, 비밀번호, 유형: 개인/사업자)
2. 사업자 선택 시 → 사업자 정보 입력 + 사업자등록증 업로드
3. Admin이 사업자 서류 검토 → 승인/반려
4. 승인된 계정으로 로그인
5. 유형에 따라 고객 페이지 / 관리자 대시보드로 분기

### 고객관리 (Admin)

1. 고객 목록 조회 (이름/이메일/전화 검색, 유형·상태 필터)
2. 고객 상세 조회 — 개인: 기본정보, 배송/결제 요약, 포인트, 등급
3. 고객 상세 조회 — 사업자: 사업자 정보, 담당자, 계약 조건, 할인율
4. 상태 변경 (활성/휴면/정지/탈퇴) + 사유 기록
5. 등급 수동 조정, 포인트 수동 지급/차감

### 직원관리 (Admin)

1. Admin이 직원 계정 생성 (이름, 이메일, 부서, 직급)
2. 사번 자동 생성 (HN + 연도 + 순번)
3. 부서 기반 기본 권한 그룹 자동 적용
4. 권한 그룹 추가/제거 (복수 선택 가능)
5. 비밀번호 초기화, 계정 비활성화

### 포인트

1. 결제 완료 시 결제 금액의 1% 자동 적립
2. 주문 시 포인트 사용 가능
3. 적립일로부터 24개월 후 자동 소멸
4. Admin이 수동 지급/차감 가능 (사유 기록)

## Business Rules

### 계정

- 아이디: 영문+숫자, 4~20자, 중복 불가
- 비밀번호: 8자 이상, 영문+숫자+특수문자 포함
- 사업자 가입 시 사업자등록증 필수 (승인 전까지 주문 불가)
- 개인은 가입 즉시 이용 가능
- 직원 계정은 Admin만 생성 가능 (최초 1명 시드)

### 고객 등급

- 월 거래량 50건 이상 → Silver
- 월 거래량 100건 이상 → Gold
- 상위 10% → VIP (수동 지정 가능)
- 매월 1일 자동 재산정

### 포인트

- 결제 금액 1% 적립
- 적립일로부터 24개월 후 소멸
- 최소 사용 단위: 100P
- 환불 시 사용 포인트 복원, 적립 포인트 차감

### 사업자 계약

- 월 100건 이상 거래 시 10% 할인 자동 적용
- 전용 물류 담당자 배정 가능
- 무료 보관일 연장 가능 (기본 30일, 최대 90일)
- 세금계산서 발행 대상

### 권한 그룹

- 관리자: 전체 권한
- 물류관리: 보관, 배송, 랙관리
- 고객관리: 고객, 결제, 문의
- 운영관리: 게시판, 통계, 로그
- 리포트: 조회, 다운로드
- 한 직원에 여러 그룹 부여 가능
- 직원 비밀번호 90일 주기 변경 권고

### 휴면/비활성화

- 고객: 180일 미접속 → 자동 휴면 (재로그인으로 복귀)
- 직원: 90일 미접속 → 자동 비활성화 (Admin 재활성화 필요)

## Status

### 고객 계정

```
pending(사업자 서류 검토) → active → dormant(180일 미접속) → active(재로그인)
                            ↗ (개인은 바로 active)
rejected(서류 반려) → pending(재제출)
active → suspended(정지) → deactivated(탈퇴)
```

### 직원 계정

```
active → inactive(90일 미접속) → active(Admin 재활성화)
active → suspended → deactivated
```

| Screen | Path | Key Features |
|--------|------|-------------|
| 고객 로그인 | /login | 아이디/비밀번호 |
| 회원가입 | /register | 유형(개인/사업자), 사업자 정보·서류 업로드 |
| 관리자 로그인 | /admin/login | 아이디/비밀번호 |

| 고객 상세 | /admin/members/:id | 개인: 기본정보·배송·결제·포인트, 사업자: +계약 조건 |
| 직원 목록 | /admin/staff | 검색(이름/이메일/부서), 상태 필터 |
| 직원 상세 | /admin/staff/:id | 기본정보, 조직정보, 권한 그룹 체크박스 |
| 직원 등록 | /admin/staff/new | 이름, 이메일, 부서, 직급, 초기 권한 |
| 마이페이지 | /mypage | 내 정보, 비밀번호 변경, 포인트 잔액, 예치금 |

## Out of Scope

- 소셜 로그인 (카카오, 네이버) — v2
- 비밀번호 찾기 (이메일/SMS) — v2
- 2단계 인증 — v2
- IP 제한, 시간 제한 접근 제어 — v2
- 조직도/보고선 시각화 — v2
- 고객 행동 패턴 분석/통계 — v2

### 4. 통관유형 자동 설정

**기본 규칙**

| 수입방식 | 기본 통관유형 | 비고 |
|----------|---------------|------|
| 개인전자상거래 | 목록통관 | 조건 만족 시 자동변경 |
| 사업자 | 일반통관 | 고정 (변경 없음) |

**개인전자상거래 선택 시 자동변경 조건 (OR 관계)**

> 아래 조건 중 **하나라도 만족**하면 **무조건 일반통관으로 자동변경**:

| 조건 | 설명 | 판단 기준 |
|------|------|-----------|
| 식품 카테고리 | 상품의 HS코드가 식품 분류에 해당 | HS코드 앞자리 기반 자동 분류 |
| 총금액 $150 초과 | 모든 상품 총금액 합계가 150달러 초과 | (단가 × 수량) 합계 > $150 |
| 단가 $150 초과 | 개별 상품의 단가가 150달러 초과 | 개별 상품 단가 > $150 |

> **주의**: 위 3가지 조건은 **OR(또는)** 관계입니다. 어느 하나만 해당되어도 일반통관으로 자동 변경됩니다.

**예시**:
- 식품 $50 × 3개 = 총 $150 → **일반통관** (식품 카테고리)
- 전자제품 $200 × 1개 = 총 $200 → **일반통관** (총금액 $150 초과 / 단가 $150 초과)
- 의류 $80 × 2개 = 총 $160 → **일반통관** (총금액 $150 초과)
- 문구용품 $30 × 3개 = 총 $90 → **목록통관** (해당 조건 없음)

**적용 방식**:
- **실시간 재계산**: 수입방식 선택 또는 상품정보 변경 시마다 통관유형 즉시 재판정
- **UI 알림**: 자동변경 발생 시 알림 표시 ("⚠️ 상품 금액/카테고리로 인해 일반통관으로 자동 변경되었습니다.")

### 5. 화물보험

- **적용 대상**: 배송대행/구매대행만 해당 (특송 운송 건만)
- **유형**: 유형 구분 없이 한 가지 유형만 존재
- **필수 가입**: 반드시 체크해야 접수 신청 가능
- **제외 서비스**: LCL 신청 / 쿠팡입고 / 재고신청 — 화물보험 미노출
- **보상 한도**: ₩300,000
- **보험료**: 상품구매 금액에 따라 청구 (API 리턴값)
- **보험 적용 제외**:
  - 천재지변 (지진, 홍수, 태풍 등)
  - 구매자의 고의 또는 중대한 과실
  - 통관 시 세관 검수에 의한 훼손
  - 포장 불량으로 인한 훼손 (보험 가입해도 보상 불가)

### 6. 약관 동의

- **전체 약관 동의** 체크 시 개별 필수 약관 모두 체크 처리
- **제출 조건**: 전체약관동의 (또는 개별 필수 약관 모두) 체크 필수
- **약관 항목**:
  - [필수] 서비스 이용약관 동의
  - [필수] 개인정보 수집 및 이용 동의
  - [필수] 화물보험 약관 동의
  - [필수] 안전거래 이용약관 동의
  - [선택] 마케팅 정보 수신 동의

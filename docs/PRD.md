# HubNext (허브넥스트)

## Problem

중국에서 상품을 구매하는 개인 소비자와 셀러가 국제 배송을 처리할 때,
판매자가 한국 직배송을 지원하지 않거나 건당 배송비가 비싸다.
중국 현지에서 상품을 수령·검수·합배송해줄 물류 인프라가 필요하지만,
개인이나 소규모 셀러가 현지 창고를 직접 운영하는 것은 현실적으로 불가능하다.

## Solution

중국 현지 물류센터에서 상품을 수령, 측정/검수, 재포장한 뒤
한국까지 배송을 대행하는 플랫폼.
고객은 웹에서 대행 신청만 하면, 입고부터 배송 완료까지 실시간으로 추적할 수 있다.

## Users

| 유형 | 설명 | 핵심 니즈 |
|------|------|----------|
| 개인 소비자 | 중국 쇼핑몰에서 직구하는 일반 사용자 | 저렴한 배송비, 상품 상태 확인, 간편한 신청 |
| 셀러 (사업자) | 중국 상품을 한국에서 판매하는 소규모 사업자 | 대량 입고, 합배송, 빠른 처리, 비용 관리 |
| Admin | 시스템 관리자 | 주문 검토, 전체 운영 관리 |
| CS팀 | 고객 서비스 담당 | 문의 대응, 반려/리턴 처리, 완료 확인 |
| QA팀 | 품질 검수 담당 | 상품 측정/검수, 리턴 상품 확인 |
| 재무팀 | 결제/정산 담당 | 결제 확인, 환불 처리, 세금계산서 |
| 물류팀 | 출고/배송 담당 | 포장, 운송장 발행, 통관 서류 |

## Core Flow

1. 고객이 웹에서 배송대행 신청
2. 시스템이 자동 접수 + 주문번호 발급
3. Admin이 신청서 검토 (승인/반려)
4. 중국 물류센터에서 상품 수령 + 사진 촬영
5. QA팀이 무게/부피 측정, 품질 검수
6. 시스템이 측정 데이터 기반 배송비 자동 계산
7. 고객에게 결제 요청 (72시간 유효)
8. 결제 확인 후 출고 처리 (포장, 운송장, 통관서류)
9. 택배사 인수 → 배송 시작
10. 실시간 배송 추적 + 고객 알림
11. 배송 완료 → CS 확인 → 후기 작성

## Goals

| 지표 | 목표 | 측정 방법 |
|------|------|----------|
| 접수~출고 처리 시간 | 결제 후 24시간 이내 출고 | 상태 전이 시간 측정 |
| 배송 완료율 | 99% | 완료 건수 / 전체 건수 |
| 예외 처리 시간 | 반려 24h, 결제실패 48h, 배송오류 72h | 에스컬레이션 모니터링 |

## Pricing

- 개인 소비자와 셀러(사업자)는 차등 요금제 적용
- 비용 구성: 기본 배송비, 부가서비스, 보험료, 세금
- VIP 등급 시스템: 등급별 혜택 (결제 유예 24시간 등)

## Payment Methods

- 카드 결제 (PG사 연동)
- 계좌이체
- 예치금 결제
- 결제 유효기간 72시간, 초과 시 자동 취소
- 결제 실패 시 24시간 내 3회 자동 재시도

## Scope

### v1 포함

- 정상 주문 처리 흐름 전체 (14단계)
- 예외 처리 4가지 (반려, 결제실패, 배송오류, 리턴)
- 고객 웹 (신청, 추적, 후기)
- 관리자 웹 (검토, 운영, 대시보드)
- 상태 관리 체계 (상태 코드 10~999)
- 실시간 알림 (이메일/SMS)
- 에스컬레이션 (4단계, 24h~96h)
- 차등 요금제 (개인/셀러)
- VIP 등급 시스템
- 예치금 결제 시스템

### v1 제외

- 모바일 앱 — 웹 반응형으로 대체
- 다국가 창고 (일본, 미국 등) — 중국 운영 검증 후 확장
- 자동 통관 시스템 — 수동 서류 처리로 시작
- 정산/수수료 관리 — 별도 시스템으로 분리 예정

## Constraints

- 중국 → 한국 단일 노선만 지원 (v1)
- 결제 유효기간 72시간 (초과 시 자동 취소)
- 리턴 신청 기한 배송 완료 후 14일 이내
- 반려 수정 기한 72시간 (초과 시 자동 취소)
- 예외 처리 이력 3년 보관

## Risks

| 리스크 | 영향 | 대응 |
|--------|------|------|
| 중국 물류센터 연동 지연 | 입고/측정 데이터 실시간성 저하 | 수동 입력 fallback 제공 |
| 택배사 API 불안정 | 배송 추적 불가 | 수동 상태 업데이트 + 복수 택배사 지원 |
| 결제 PG사 장애 | 결제 처리 중단 | 자동 재시도 3회 + 대체 결제 수단 |
| 금지품목 입고 | 법적 리스크, 배송 지연 | 검수 단계에서 필터링 + 즉시 반려 |

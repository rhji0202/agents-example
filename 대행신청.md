# 대행신청 (배송대행/구매대행)

## 개요
- **액션명**: 대행신청
- **카테고리**: 주문 관리
- **우선순위**: 매우 높음

## 기능 목적
배송대행과 구매대행 모두 동일한 신청서 페이지에서 진행. 접근버튼은 나뉘어져 있음.

## 주요 기능 상세

### 1. 접근 방식 (5가지 경로)
| 접근 버튼 | 경로 | 설명 |
|-----------|------|------|
| 🚚 배송대행 | 대행신청 | 일반 배송대행 서비스 |
| 🛒 구매대행 | 대행신청 | 구매대행 서비스 (동일 구조) |
| 📦 LCL 신청 | LCL_쿠팡입고 | LCL 신청서 (별도 구조) |
| 🏪 쿠팡입고 | LCL_쿠팡입고 | 쿠팡입고 신청서 (별도 구조) |
| 📋 재고신청 | 재고신청 | 재고신청서 (별도 구조) |

- **배송대행/구매대행**: 동일한 신청서 페이지 (신청유형만 다름)
- **LCL/쿠팡입고/재고신청**: 각 서비스 전용 신청서 페이지

### 2. One-Page 전체 보기
- **스크롤 방식**: 위에서 아래로 모든 입력 항목 표시
- **그룹 분리**: 구역별로 구역 분리 (물류/수취인/상품/요청사항/보험/결제/약관)
- **임시저장**: 입력 중 임시 저장 가능

### 3. 물류센터, 운송방식, 수입방식

#### 물류센터
| 코드 | 이름 | 지역 |
|------|------|------|
| WH001 | 위해 | 중국 |
| WH002 | 위해-인천1 | 중국-인천 |
| WH003 | 위해-인천2 | 중국-인천 |
| WH004 | 일본 | 일본 |

#### 운송방식
| 코드 | 이름 | 소요기간 |
|------|------|----------|
| SEA | 해운 | 14-30일 |
| AIR | 항공 | 3-7일 |

#### 수입방식
| 코드 | 이름 | 통관유형 |
|------|------|----------|
| PCE | 개인전자상거래 | 목록통관 |
| BS | 사업자 | 일반통관 |

- **통관유형 자동 설정**: 수입방식 선택 시 자동 설정

### 4. 수취인정보 입력
- **주소록 불러오기**: 저장된 배송지 목록에서 선택
- **기본배송지 사용**: 체크 시 저장된 기본 주소 자동 입력
- **통관유형**: 개인통관 / 사업자통관 선택
  - 개인통관: 개인통관고유부호 입력
  - 사업자통관: 사업자등록번호, 마킹번호 입력

### 5. 상품정보 입력 (상세)

#### 기본 정보
| 필드 | 설명 |
|------|------|
| 통관품목 검색 | 키워드 검색 후 HS코드 자동 입력 |
| 상품명 (영문) | 상품명 영문 입력 |
| 상품명 (중문) | 상품명 중문 입력 |
| 모델명/재질/규격/사이즈 | 선택적 입력 |
| 옵션정보 | 색상, 상품 단위 |
| 단가 × 수량 | 금액 자동 계산 |

#### 배송/통관 정보
| 필드 | 설명 |
|------|------|
| 순량 | 순중량 (kg) 입력 |
| 상품URL | 상품 페이지 URL |

#### 이미지/파일
| 필드 | 설명 |
|------|------|
| 이미지URL | 이미지 URL 직접 입력 |
| 이미지 업로드 | 파일 선택/드래그 앤 드롭 (최대 5개) |
| 리뷰이벤트 PDF | PDF 파일만 업로드 가능 |

#### 트래킹번호 (멀티 입력)
```
🔖 트래킹번호: [+ 추가]
  └─ ☑ 추후 입력 가능 (체크 시 제출 가능)
  └─ [1] [운송사 ▼] [트래킹번호 입력____________] [삭제]
  └─ [2] [운송사 ▼] [____________________________] [삭제]
```
- **추후 입력 가능**: 체크 시 트래킹번호 없이 제출 가능
- **멀티 입력**: 여러 개의 트래킹번호 추가 가능

### 6. 통관유형 자동 설정
| 수입방식 | 조건 | 통관유형 |
|----------|------|----------|
| 개인전자상거래 | 기본 | 목록통관 |
| 개인전자상거래 | **상품이 식품카테고리** OR **총금액/단가 $150 초과** 시 | **일반통관 (자동변경)** |
| 사업자 | 기본 | 일반통관 |

#### 자동 변경 규칙 상세

**개인전자상거래 선택 시 기본값**: 목록통관
→ 단, 아래 조건 중 **하나라도 만족**하면 **무조건 일반통관으로 자동변경**:

| 조건 | 설명 | 판단 기준 |
|------|------|-----------|
| 식품 카테고리 | 상품의 HS코드가 식품 분류에 해당 | HS코드 앞자리 기반 자동 분류 |
| 총금액 $150 초과 | 모든 상품 총금액 합계가 150달러 초과 | (단가 × 수량) 합계 > $150 |
| 단가 $150 초과 | 개별 상품의 단가가 150달러 초과 | 개별 상품 단가 > $150 |

> **주의**: 위 3가지 조건은 **OR(또는)** 관계입니다. 어느 하나만 해당되어도 일반통관으로 자동 변경됩니다.
>
> **예시**:
> - 식품 $50 × 3개 = 총 $150 → **일반통관** (식품 카테고리)
> - 전자제품 $200 × 1개 = 총 $200 → **일반통관** (총금액 $150 초과 / 단가 $150 초과)
> - 의류 $80 × 2개 = 총 $160 → **일반통관** (총금액 $150 초과)
> - 문구용품 $30 × 3개 = 총 $90 → **목록통관** (해당 조건 없음)

- **자동 설정**: 수입방식/상품정보 입력 시 실시간으로 통관유型 재계산 및 적용
- **UI 안내**: 통관유형이 자동 변경될 경우 사용자에게 알림 표시 ("⚠️ 상품 금액/카테고리로 인해 일반통관으로 자동 변경되었습니다.")

### 9. 화물보험 (가입하기 & 내용확인 - 필수 가입)
- **체크 방식**: ☐ 가입하기 & 내용확인 체크박스
- **클릭 시**: 레이어팝업 노출 (보험 상세내용 확인 가능)
- **유형 구분**: 없음 (한 가지 유형만 존재)
- **필수 가입**: 반드시 체크해야 접수 신청 가능
- **보험료 안내**: 상품구매 금액에 따라 보험료가 청구됨

#### 보험 유형 상세 (유형 구분 없음)
| 옵션 | 요금 | 보상 한도 | 설명 |
|------|------|-----------|------|
| 기본 보험 | 유료 (상품구매 금액에 따라 청구) | ₩300,000 | 유형 구분 없이 한 가지 유형만 제공 |

#### 보험 적용 제외 사항
- 천재지변 (지진, 홍수,台风 등)
- 구매자의 고의 또는 중대한 과실
- 통관 시 세관 검수에 의한 훼손
- 포장 불량으로 인한 훼손 (보험 加入해도 보상 불가)

### 10. 예치금 자동결제
| 옵션 | 설명 |
|------|------|
| 자동결제 처리 | 결제 시 예치금에서 자동 차감 |
| 미처리 | 결제 시 직접 처리 (카드 등) |

### 11. 약관동의
- **모든 필수 약관 동의**: 체크 시 모든 약관에 동의 처리
- **제출 조건**: 전체약관동의 체크 필수

| 수입방식 | 기본 통관유형 | 자동변경 조건 → 일반통관 | 비고 |
|----------|---------------|----------------------|------|
| 개인전자상거래 | 목록통관 | 식품카테고리 OR 총금액¥1,000초과 OR 단가¥1,000초과 | OR조건, 해당 시 무조건 변경 |

### 5. 화물보험
- **적용**: 배송대행/구매대행만 가능
- **유형**: 유형 구분 없이 한 가지 유형만 존재
- **필수**: 필수 가입
- **제외**: LCL/쿠팡입고/재고신청

### 6. 약관 동의
- **모든 약관 동의** 체크 시 개별 약관도 모두 체크 처리
- **제출 조건**: 전체약관동의 체크 필수

### 7. 고객사서함 저장
- **저장 시점**: 접수 신청 완료 시
- **저장 위치**: 고객사서함
- **저장 내용**: 신청서 전체 정보
- **접근**: 마이페이지 > 고객사서함에서 조회 가능

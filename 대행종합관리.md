# 대행종합관리 (Admin - 대행 주문 통합 관리)

## 개요
- **액션명**: 대행종합관리
- **카테고리**: 관리자 - 대행관리
- **우선순위**: 매우 높음
- **버전**: 5.1.1

## 기능 목적
관리자가 한 페이지에서 모든 대행 주문(배송대행/구매대행/특송/LCL/쿠팡입고/재고)을 조회, 처리, 관리하고, 창고작업자와의 협업까지 가능하는 통합 대시보드.

#### 주문 행 구성 규칙

**1. 주문 헤더 줄 (테이블 필터명 상단 표시)**
- 테이블 필터명은 주문 헤더 줄 위에 한 번만 표시됨
- 개별 주문 행에는 필터명 없이 값들만 표시됨

```
[□] [▶] APP2025040001 🟢 입고완료 [합배송] [주문상세보기]
```
- □: 체크박스 (다중 선택용)
- ▶: 펼치기/닫기 아이콘 (▼/▶)
- APP2025040001: 주문번호
- 🟢 입고완료: 주문상태 (아이콘 + 상태명)
- [합배송]: 합배송/나눔배송 여부 표시
- [주문상세보기]: 주문 상세 보기 버튼

**2. 주문 정보 테이블 줄**
```
| 배송대행/인천1/항공 | 홍길동(mail@hubnext.com)/김철수(개인통관) | 묶음 123123123 | 3/3 | 5개/￥100（＄150） | 배송비용/₩4,010,000 | CJ123456789KR/2026-04-10/123-45-67890 | 입고완료/next10 | 2026-04-01/2026-04-07/2026-04-12 |
```

**3. 추가 정보 영역** (2번-10번칸 넓이만큼의 통합 영역에서 새로운 테이블 구조로 출력)

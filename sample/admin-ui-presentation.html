Flow

회원가입 & 로그인

개인 고객

1
아이디/비밀번호 입력
2
가입 즉시 active
3
로그인 → 고객 페이지

사업자 고객

1
사업자 정보 입력
2
사업자등록증 업로드
3
Admin 검토 (pending)
4
승인 → active / 반려 → rejected

로그인 규칙

  • • 아이디: 영문+숫자, 4~20자, 중복 불가
  • • 비밀번호: 8자 이상, 영+숫+특수
  • • 유형별 분기: 고객 → 고객페이지, 직원 → 대시보드
  • • 사업자: 승인 전까지 주문 불가

휴면 & 비활성화

  • 고객 180일 미접속 → 자동 휴면
  • → 재로그인으로 즉시 복귀
  • 직원 90일 미접속 → 자동 비활성화
  • → Admin 재활성화 필요

직원 계정 생성

1
Admin이 생성 (자가가입 불가)
2
사번 자동: HN + 연도 + 순번
3
부서 기반 권한 자동 적용

Business Rules

등급 & 포인트

고객 등급 산정

Normal 기본
Silver 월 50건+
Gold 월 100건+
VIP 상위 10% (수동 가능)

매월 1일 자동 재산정

PointTransaction 모델

type: earn / use / expire / admin_add / admin_deduct
amount: int (양수)
balance_after: 거래 후 잔액
expires_at: 소멸 예정일 (earn 시)

포인트 규칙

  • 1
    결제 금액의 1% 자동 적립
  • 2
    적립일로부터 24개월 후 소멸
  • 3
    최소 사용 단위: 100P
  • 4
    주문 시 포인트 사용 가능

환불 시 포인트

  • • 사용 포인트 → 복원
  • • 적립 포인트 → 차감

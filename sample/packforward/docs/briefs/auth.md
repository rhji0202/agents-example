# Auth Brief

## Summary
소비자가 회원가입하면 개인 미국 창고 주소를 발급받고, 통관부호를 등록한다.

## Flow
1. 이메일/비밀번호로 회원가입
2. 가입 즉시 개인 창고 주소 자동 발급 (US-00001)
3. 마이페이지에서 창고 주소 확인 + 복사
4. 개인통관부호 등록 (선택, 배송 신청 시 필수)
5. 한국 배송지 주소 등록 (선택, 결제 시 필수)

## Business Rules
- 창고 주소 형식: 실제 미국 주소 + Suite/Apt에 개인 코드 (US-00001)
- 개인통관부호: P + 12자리 숫자, 관세청 API로 실시간 검증
- 통관부호 검증 실패 시 저장 불가 (에러 메시지 표시)
- 소셜 로그인은 v1에서 제외 (이메일만)
- 비밀번호 찾기는 Supabase Auth 기본 기능 사용

## Status
없음 (User는 stateless)

## Screens
| Screen | Path | Key Features |
|--------|------|-------------|
| 회원가입 | /signup | 이메일, 비밀번호, 이름, 연락처 |
| 로그인 | /login | 이메일, 비밀번호 |
| 내 창고주소 | /my/warehouse | 주소 표시, 복사 버튼, 사용법 안내 |
| 통관부호 등록 | /my/customs | 입력 + 실시간 검증 + 저장 |
| 내 정보 | /my/profile | 이름, 연락처, 한국 배송지 수정 |

## Out of Scope
- 소셜 로그인 (Google, Kakao) — v2
- 휴대폰 본인인증 — v2
- 다중 배송지 관리 — v1은 1개만

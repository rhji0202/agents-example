# Inbound Brief

## Summary
해외 쇼핑몰에서 창고로 도착한 상품을 등록하고, 검수 후 소비자에게 알린다.

## Flow
1. 소비자가 입고 예정 등록 (쇼핑몰명, 운송장, 예상 품목)
2. 택배가 창고에 도착
3. 창고 직원이 운송장 번호로 소비자 매칭
4. 검수: 외관 확인, 사진 2장 이상 촬영
5. 무게/크기 측정 입력
6. 입고 완료 → 소비자에게 이메일 알림

## Business Rules
- 사전등록 없어도 창고 주소의 Suite 번호로 자동 매칭
- 사전등록이 있으면 운송장 번호로 매칭 (더 정확)
- 검수 사진 최소 2장 (외관 전면, 라벨)
- 무료 보관 30일, 31일부터 건당 $1/일
- 60일 초과 시 폐기 경고 이메일
- 90일 초과 시 자동 폐기 (사전 알림 필수)
- 금지품목 발견 시 rejected 처리 → 소비자에게 알림 → 반송 or 폐기 선택

## Status
arriving → received → inspected → stored → consolidated
                                 ↘ rejected
